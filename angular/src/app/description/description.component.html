Tema projekta: Chat bot na našem jeziku.

Aplikacija zapravo daje mogućnost da se bira chatbot iz date liste. Mi smo do sad implementirali samo jednog bota, koji priča na našem jeziku. Bot nije specijalizovan za konkretnu namjenu, već podržava opšte razgovore.

Stek tehnologija koje smo koristili:

  • Rasa X framework za pravljenje chat botova
  • MySQL baza podataka
  • Flask Web framework za back-end
  • Angular front-end framework
